# Build

This is the home for Opentecture's Build Project.

The initial goal is design a modular home
at model-scale
using 3d printing methods.

### Components

* Structural:
  * Foundation
  * Exterior Walls
  * Utilities/Systems:
    * Water
    * Electricity
    * Network
    * Plumbing/Sewage
    * Air Conditioning/HVAC
* Functional:
  * Interior Walls
  * Doors
* Aesthetic:
  * Decor
  * Finishing:
    * Door Knobs
